wsrc aims to need almost no configuration. As you add settings,
document each one here with its default.

## Global flags

| Flag | Meaning |
|---|---|
| `--help` | Help for any command |
| `--version` | Print the version |

## Environment variables

None yet. Add them here as the tool grows, prefixed `wsrc_` in upper
case (for example `wsrc_TOKEN`).
